## üîå Backend Technologies

### **Elysia**

- **TypeScript-First** - Built with TypeScript from the ground up
- **High Performance** - Built on Bun's HTTP server
- **Plugin System** - Extensible architecture with plugins
- **oRPC Integration** - Native support for type-safe RPC
- **WebSocket Support** - Real-time communication capabilities

**Purpose**: High-performance web framework for building APIs

### **oRPC (Object Remote Procedure Call)**

- **Type Safety** - End-to-end type safety between client and server
- **Automatic Validation** - Runtime type checking and validation
- **Zero Configuration** - Works out of the box with TypeScript
- **Multiple Transports** - HTTP, WebSocket, and more
- **Code Generation** - Automatic client code generation

**Purpose**: Type-safe remote procedure call protocol

### **Drizzle ORM**

- **Type-Safe Queries** - Compile-time SQL validation
- **Zero Runtime Overhead** - No query builders or runtime overhead
- **Migration System** - Automatic schema evolution
- **Multiple Databases** - SQLite, PostgreSQL, MySQL support
- **Relations** - Type-safe relationship handling

**Purpose**: Type-safe SQL toolkit and ORM
